Key State Variables

text_input_queue — FIFO buffer (max 3), fed by sense_data_callback, consumed by _process_text_input

execution_paused — gates the main loop; auto-cleared by goal commands and User/agent input

awaiting_ask_response — when true, the ask primitive polls text_input_queue directly, so _process_text_input skips

interrupt_requested — set by stop/interrupt commands; checked at step boundaries by planner

continuous_mode — NOT cleared by stop; user must toggle explicitly

Dispatch Priority (User source)

Direct JSON action → goal: → task: → proceed → terminate → reuse/unblock/tasks → end phrase → dialog-with-agent → regular conversation (envision + goal)
